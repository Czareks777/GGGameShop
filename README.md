GGameShop



Opis: Aplikacja napisana została na zaliczeniu przedmiotu ASP.NET w grupie 3 osobowej. 



Cel: Sprzedaż gier komputerowych na platformie internetowej.


Technologie: ASP.NET MVC, HTML 5, CSS.
